# ประกาศองค์การบริหารส่วนตำบลเขาน้อย เรื่อง โครงสร้างและการจัดองค์กรในการดำเนินงาน สรุปอำนาจหน้าที่ที่สำคัญ วิธีดำเนินงานและสถานที่ติดต่อเพื่อขอรับข้อมูลข่าวสารขององค์การบริหารส่วนตำบลเขาน้อย

ประกาศองค์การบริหารส่วนตําบลเขาน้อย เรื่อง โครงสร้างและการจัดองค์กรในการดําเนินงาน สรุปอํานาจหน้าที่ที่สําคัญ วิธีดําเนินงาน และสถานที่ติดต่อเพื่อขอรับข้อมูลข่าวสารขององค์การบริหารส่วนตําบลเขาน้อย ตามพระราชบัญญัติข้อมูลข่าวสารของราชการ พ . ศ . 2540 ได้ประกาศในราชกิจจานุเบกษา เมื่อวันที่ 10 กันยายน 2540 และมีผลใช้บังคับตั้งแต่วันที่ 9 ธันวาคม 2540 เป็นต้นมา โดยพระราชบัญญัติดังกล่าวได้กําหนดให้หน่วยงานของรัฐรวบรวมข้อมูลข่าวสารของราชการ ตามมาตรา 7 ( 1 ) - ( 3 ) ที่มีผลใช้บังคับอยู่ ได้แก่ โครงสร้าง อํานาจหน้าที่ วิธีดําเนินงาน และสถานที่ติดต่อลงพิมพ์ในราชกิจจานุเบกษา นั้น เพื่อให้เป็นไปตามมาตรา 7 ( 1 ) ( 2 ) และ ( 3 ) แห่งพระราชบัญญัติข่าวสารของราชการ พ . ศ . 2540 ที่จะให้ประชาชนมีสิทธิเข้าถึงและได้รับข้อมูลข่าวสารเกี่ยวกับการดําเนินงานต่าง ๆ ของทางราชการและสามารถแสดงความคิดเห็นและใช้สิทธิทางการเมืองได้โดยถูกต้องกับสภาพความเป็นจริง จึงสมควรประกาศโครงสร้างและการจัดองค์กรในการดําเนินงานขององค์การบริหารส่วนตําบลเขาน้อย และสถานที่ติดต่อเพื่อขอรับข้อมูลข่าวสาร ดังต่อไปนี้ การจัดโครงสร้างองค์กรในการดําเนินการ อํานาจหน้าที่ที่สําคัญขององค์การบริหารส่วนตําบลเขาน้อย เป็นไปตามพระราชบัญญัติสภาตําบลและองค์การบริหารส่วนตําบล พ . ศ . 2537 และแก้ไขเพิ่มเติม พระราชบัญญัติกําหนดแผนและขั้นตอนการกระจายอํานาจให้แก่องค์กรปกครองส่วนท้องถิ่น พ . ศ . 2542 และที่แก้ไขเพิ่มเติม พระราชบัญญัติระเบียบบริหารงานบุคคลส่วนท้องถิ่น พ . ศ . 2542 และกฎหมายอื่น ๆ ที่เกี่ยวข้อง ข้อ 1 โครงสร้างและการจัดองค์กรในการดําเนินงานขององค์การบริหารส่วนตําบลเขาน้อย ดังนี้ ( 1 ) สภาองค์การบริหารส่วนตําบล ประกอบด้วย ( 1 . 1 ) ประธานสภา ( 1 . 2 ) รองประธานสภา ( 1 . 3 ) สมาชิกสภาองค์การบริหารส่วนตําบล ( 1 . 4 ) เลขานุการสภาองค์การบริหารส่วนตําบล ้ หนา 301 ่ เลม 139 ตอนพิเศษ 300 ง ราชกิจจานุเบกษา 23 ธันวาคม 2565

( 2 ) ฝ่ายบริหาร ประกอบด้วย ( 2 . 1 ) นายกองค์การบริหารส่วนตําบล ( 2 . 2 ) รองนายกองค์การบริหารส่วนตําบล ( 2 . 3 ) เลขานุการนายกองค์การบริหารส่วนตําบล ( 3 ) ส่วนราชการองค์การบริหารส่วนตําบลเขาน้อย ประกอบด้วย ( 3 . 1 ) สํานักปลัดองค์การบริหารส่วนตําบล รับผิดชอบงานด้าน ( 3 . 1 . 1 ) งานบริหารทั่วไป ( 3 . 1 . 2 ) งานด้านสวัสดิการสังคมสงเคราะห์ ( 3 . 1 . 3 ) งานนโยบายและแผน ( 3 . 1 . 4 ) งานป้องกันและบรรเทาสาธารณภัย ( 3 . 1 . 5 ) งานสาธารณสุขและสิ่งแวดล้อม ( 3 . 2 ) กองคลัง รับผิดชอบงานด้าน ( 3 . 2 . 1 ) งานการเงินและบัญชี ( 3 . 2 . 2 ) งานพัฒนารายได้ ( 3 . 2 . 3 ) งานทะเบียนทรัพย์สินและพัสดุ ( 3 . 3 ) กองช่าง รับผิดชอบงานด้าน ( 3 . 3 . 1 ) งานบริหารทั่วไป ( 3 . 3 . 2 ) งานก่อสร้าง ( 3 . 3 . 3 ) งานบริการสาธารณูปโภค ( 3 . 3 . 4 ) งานสถานที่และไฟฟ้าสาธารณะ ( 3 . 4 ) กองการศึกษา ศาสนาและวัฒนธรรม รับผิดชอบงานด้าน ( 3 . 4 . 1 ) งานบริหารทั่วไป ( 3 . 4 . 2 ) งานจัดการศึกษาศูนย์พัฒนาเด็กเล็ก ( 3 . 4 . 3 ) งานส่งเสริมการศึกษา ศาสนา วัฒนธรรม กีฬา และนันทนาการ ( 3 . 5 ) หน่วยตรวจสอบภายใน ( 3 . 5 . 1 ) งานตรวจสอบภายใน ้ หนา 302 ่ เลม 139 ตอนพิเศษ 300 ง ราชกิจจานุเบกษา 23 ธันวาคม 2565

ข้อ 2 อํานาจหน้าที่ขององค์การบริหารส่วนตําบลเขาน้อย ประกอบด้วย 2 . 1 อํานาจหน้าที่ตามมาตรา 67 แห่งพระราชบัญญัติสภาตําบลและองค์การ บริหารส่วนตําบล พ . ศ . 2537 และที่แก้ไขเพิ่มเติม กําหนดให้องค์การบริหารส่วนตําบลมีอํานาจ หน้าที่ดําเนินการภายในเขตองค์การบริหารส่วนตําบล ดังนี้ ( 1 ) จัดให้มีการบํารุงรักษาทางน้ําและทางบก ( 2 ) รักษาความสะอาดของถนน ทางน้ํา ทางเดิน และที่สาธารณะ รวมทั้ง กําจัดมูลฝอยและสิ่งปฏิกูล ( 3 ) ป้องกันโรคและระงับโรคติดต่อ ( 4 ) ป้องกันและบรรเทาสาธารณภัย ( 5 ) ส่งเสริมการศึกษา ศาสนา และวัฒนธรรม ( 6 ) ส่งเสริมการพัฒนาสตรี เด็ก เยาวชน ผู้สูงอายุ และผู้พิการ ( 7 ) คุ้มครองดูแล และบํารุงรักษาทรัพยากรธรรมชาติและสิ่งแวดล้อม ( 8 ) บํารุงรักษาศิลปะ จารีตประเพณี ภูมิปัญญาท้องถิ่น และวัฒนธรรมอันดี ของท้องถิ่น ( 9 ) ปฏิบัติหน้าที่อื่นตามที่ทางราชการมอบหมาย โดยจัดสรรงบประมาณ หรือบุคลากรให้ตามความจําเป็นและสําคัญ 2 . 2 อํานาจหน้าที่ตามมาตรา 68 แห่งพระราชบัญญัติข่าวสารของราชการ พ . ศ . 2540 และที่แก้ไขเพิ่มเติม กําหนดให้องค์การบริหารส่วนตําบลเขาน้อยอาจจัดทํากิจการ ในเขตองค์การบริหารส่วนตําบล ดังต่อไปนี้ ( 1 ) ให้มีน้ําเพื่อการอุปโภค บริโภค และการเกษตร ( 2 ) ให้มีและบํารุงการไฟฟ้าหรือแสงสว่างโดยวิธีอื่น ( 3 ) ให้มีและบํารุงรักษาทางระบายน้ํา ( 4 ) ให้มีและบํารุงสถานที่ประชุม การกีฬา การพักผ่อนหย่อนใจและสวนสาธารณะ ( 5 ) ให้มีการส่งเสริมกลุ่มเกษตรกรและกิจการสหกรณ์ ( 6 ) ส่งเสริมให้มีอุตสาหกรรมในครอบครัว ้ หนา 303 ่ เลม 139 ตอนพิเศษ 300 ง ราชกิจจานุเบกษา 23 ธันวาคม 2565

( 7 ) บํารุงและส่งเสริมการประกอบอาชีพของราษฎร ( 8 ) การคุ้มครองดูแลและรักษาทรัพย์สินอันเป็นสาธารณสมบัติของแผ่นดิน ( 9 ) หาผลประโยชน์จากทรัพย์สินขององค์การบริหารส่วนตําบล ( 10 ) ให้มีตลาด ท่าเทียบเรือ และท่าข้าม ( 11 ) กิจการเกี่ยวกับการพาณิชย์ ( 12 ) การท่องเที่ยว ( 13 ) การผังเมือง 2 . 3 อํานาจหน้าที่ตามมาตรา 16 แห่งพระราชบัญญัติกําหนดแผนและขั้นตอน การกระจายอํานาจให้แก่องค์กรปกครองส่วนท้องถิ่น พ . ศ . 2542 และที่แก้ไขเพิ่มเติม กําหนดให้ องค์การบริหารส่วนตําบลมีอํานาจและหน้าที่ในการจัดระบบบริการสาธารณะเพื่อประโยชน์ของประชาชน ในท้องถิ่นของตนเอง ดังนี้ ( 1 ) การจัดทําแผนพัฒนาท้องถิ่นของตนเอง ( 2 ) การจัดให้มี และบํารุงรักษาทางบกทางน้ํา และทางระบายน้ํา ( 3 ) การจัดให้มีและควบคุมตลาด ท่าเทียบเรือ ท่าข้าม และที่จอดรถ ( 4 ) การสาธารณูปโภค และการก่อสร้างอื่น ๆ ( 5 ) การสาธารณูปการ ( 6 ) การส่งเสริม การฝึก และการประกอบอาชีพ ( 7 ) การพาณิชย์ และการส่งเสริมการสงทุน ( 8 ) การส่งเสริมการท่องเที่ยว ( 9 ) การจัดการศึกษา ( 10 ) การสังคมสงเคราะห์ และการพัฒนาคุณภาพชีวิตเด็ก สตรี คนชราและผู้ด้อยโอกาส ของท้องถิ่น ( 11 ) การบํารุงรักษาศิลปะ จารีตประเพณี ภูมิปัญญาท้องถิ่น และวัฒนธรรมอันดี ( 12 ) การปรับปรุงแหล่งชุมชนแออัด และการจัดการเกี่ยวกับที่อยู่อาศัย ( 13 ) การจัดให้มี และบํารุงรักษาสถานที่พักผ่อนหย่อนใจ ( 14 ) การส่งเสริมกีฬา ้ หนา 304 ่ เลม 139 ตอนพิเศษ 300 ง ราชกิจจานุเบกษา 23 ธันวาคม 2565

( 15 ) การส่งเสริมประชาธิปไตย ความเสมอภาค และสิทธิเสรีภาพของประชาชน ( 16 ) ส่งเสริมการมีส่วนร่วมของราษฎรในการพัฒนาท้องถิ่น ( 17 ) การรักษาความสะอาด และความเป็นระเบียบเรียบร้อยของบ้านเมือง ( 18 ) การกําจัดมูลฝอย สิ่งปฏิกูล และน้ําเสีย ( 19 ) การสาธารณสุข การอนามัยครอบครัว และการรักษาพยาบาล ( 20 ) การจัดให้มี และควบคุมสุสาน และฌาปนสถาน ( 21 ) การควบคุมการเลี้ยงสัตว์ ( 22 ) การจัดให้มี และควบคุมการฆ่าสัตว์ ( 23 ) การักษาความปลอดภัย ความเป็นระเบียบเรียบร้อย และการอนามัย โรงมหรสพ และสาธารณสถานอื่น ๆ ( 24 ) การจัดการการบํารุงรักษา และการใช้ประโยชน์จากป่าไม้ ที่ดิน ทรัพยากรธรรมชาติ และสิ่งแวดล้อม ( 25 ) การผังเมือง ( 26 ) การขนส่ง และการวิศวกรรมจราจร ( 27 ) การดูแลรักษาที่สาธารณะ ( 28 ) การควบคุมอาคาร ( 29 ) การป้องกันและบรรเทาสาธารณภัย ( 30 ) การรักษาความสงบเรียบร้อย การส่งเสริมและสนับสนุนการป้องกันรักษา ความปลอดภัยในชีวิตและทรัพย์สิน ( 31 ) กิจการอื่นใด ที่เป็นผลประโยชน์ของประชาชนในท้องถิ่นตามที่คณะกรรมการ ประกาศกําหนด ข้อ 3 วิธีการดําเนินงานขององค์การบริหารส่วนตําบลเขาน้อย การบริหารราชการขององค์การบริหารส่วนตําบลเขาน้อย จะต้องเป็นไปเพื่อประโยชน์สุข ของประชาชน เกิดผลสัมฤทธิ์ต่อภารกิจภาครัฐ ความมีประสิทธิภาพ ความคุ้มค่าในเชิงภารกิจแห่งรัฐ การลดขั้นตอนการปฏิบัติงาน การอํานวยความสะอาดและตอบสนองความต้องการของประชาชน ทั้งนี้ ให้สอดคล้องกับแผนพัฒนาจังหวัด แผนพัฒนาอําเภอ แผนพัฒนาตําบล นโยบายของรัฐบาล ้ หนา 305 ่ เลม 139 ตอนพิเศษ 300 ง ราชกิจจานุเบกษา 23 ธันวาคม 2565
